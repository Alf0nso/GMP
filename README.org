* Monadic Parsing in Haskell

the web related to Haskell's Parser Monads always goes back to the
paper: [[https://www.cs.nott.ac.uk/~pszgmh/pearl.pdf][Monadic Parsing in Haskell]] (by Graham Hutton and Erik Meijer).
